# Alerta Sofia

Proyecto con frontend (Vite + React) y backend (Express) para una simulacion de radio dinamico.

## Requisitos

- Node.js 18+ (recomendado)
- npm

## Instalacion

Desde la raiz del repo:

```
npm run install:all
```

## Ejecucion en desarrollo

En dos terminales separadas:

```
npm run dev:backend
```

```
npm run dev:frontend
```

## Scripts utiles

- `npm run install:all` instala dependencias en backend y frontend
- `npm run dev:backend` levanta el servidor backend
- `npm run dev:frontend` levanta el servidor frontend

## Estructura

```
backend/   API y logica de simulacion
frontend/  UI web (Vite + React)
poc_libs/  librerias POC
```
